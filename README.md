# McMasterful Bookstore — Docker API

A TypeScript REST API containerized with Docker, featuring warehouse and 
order management with full test coverage via Vitest. Built on the REST API 
foundation with Docker-based development and deployment.

## Tech Stack

- **Language:** TypeScript
- **Testing:** Vitest
- **Containerization:** Docker / Dev Container
- **CI:** GitHub Actions

## Project Structure

- `src/` — API source code including warehouse and orders implementation
- `adapter/` — Data access layer

## Getting Started

Open in VS Code Dev Container, or run with Docker:

```bash
npm install
npm test
```
